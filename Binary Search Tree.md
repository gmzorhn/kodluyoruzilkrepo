**Proje 3: Binary Search Tree
Verilen Dizi: [7, 5, 1, 8, 3, 6, 0, 9, 4, 2]**

*Dizinin en başındaki eleman Root (Kök) kabul edilir ve diğer elemanlar sırayla ağaca eklenir.*

Root: 7'dir.

5 Ekleniyor: 7'den küçük -> Root'un soluna eklenir.

1 Ekleniyor: 7'den küçük (sola gir), 5'ten küçük -> 5'in soluna eklenir.

8 Ekleniyor: 7'den büyük -> Root'un sağına eklenir.

3 Ekleniyor: 7'den küçük (sola gir), 5'ten küçük (sola gir), 1'den büyük -> 1'in sağına eklenir.

6 Ekleniyor: 7'den küçük (sola gir), 5'ten büyük -> 5'in sağına eklenir.

0 Ekleniyor: 7'den küçük (sola gir), 5'ten küçük (sola gir), 1'den küçük -> 1'in soluna eklenir.

9 Ekleniyor: 7'den büyük (sağa gir), 8'den büyük -> 8'in sağına eklenir.

4 Ekleniyor: 7'den küçük (sola gir), 5'ten küçük (sola gir), 1'den büyük (sağa gir), 3'ten büyük -> 3'ün sağına eklenir.

2 Ekleniyor: 7'den küçük (sola gir), 5'ten küçük (sola gir), 1'den büyük (sağa gir), 3'ten küçük -> 3'ün soluna eklenir.
